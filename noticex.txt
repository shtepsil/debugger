Обновление проекта в репозитории:

git add . - добавить изменённые файлы в git
git status - покажет какие файлы были внесены в git память
git push -u origin main - обновить файлы в репозитории

==========================================================================
Некоторые Команды

git init
git remote add origin https://github.com/shtepsil/project.git
git config checkout.defaultRemote origin
git pull origin main
